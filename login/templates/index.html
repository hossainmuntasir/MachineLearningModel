About the Project

This project uses Machine Learning to predict the fan status in three different buildings, using a year's worth of data to optimize energy efficiency. By analyzing features such as temperature, CO2 levels, humidity, and damper status, the model predicts when the fan should be turned on or off to minimize unnecessary energy consumption while maintaining indoor comfort. The predictive model is trained on historical data from each building, helping to automate fan control and allowing building managers to make data-driven decisions that improve sustainability and overall operational efficiency.

Model Evaluation

The "Model Evaluation" page provides a comprehensive overview of the machine learning model's performance used in optimizing building efficiency. It includes visualizations such as feature importance, which shows which factors have the most significant impact on the predictions, and a confusion matrix, which measures the accuracy of the model in predicting fan operation.

Additionally, the page includes a scatter plot of building data that visualizes the relationship between actual and predicted fan statuses across different zones. Users can explore these predictions by interacting with filter options, allowing them to select specific zones, dates, and time ranges. The filters help users easily analyze patterns in fan usage, identify misclassifications, and better understand the factors influencing energy savings across various areas of the building.
